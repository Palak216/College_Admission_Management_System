# College Admission Management System

## 📌 Project Overview

The College Admission Management System is a SQL-based database project designed to manage student admissions efficiently. The database stores information about students, departments, applications, documents, and fee payments.

## 🛠️ Technologies Used

* MySQL
* SQL

## 📂 Database Tables

1. Students
2. Departments
3. Applications
4. Documents
5. FeePayments

## ✨ Features

* Student Information Management
* Department Management
* Admission Application Tracking
* Document Verification Status
* Fee Payment Management
* SQL Queries using Joins

## 🗄️ Database Schema

The database consists of five related tables connected using primary and foreign keys.

## 📊 Implemented Queries

* Student admission details
* Application reports
* Fee payment reports
* Document verification reports
* Approved admissions list
* Complete admission status reports

## 🚀 How to Run

1. Create a new database in MySQL.
2. Execute `create_tables.sql`.
3. Execute `insert_data.sql`.
4. Run the query files inside the `queries` folder.
